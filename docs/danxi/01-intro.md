# 应用简介

旦夕 App 是一个复旦综合服务应用，旨在为复旦学子提供更便捷的校园生活服务。

旦夕 App 使用 Flutter 开发，支持 Windows、macOS（M1）、Linux、Android、iOS 等多个平台。
